HTML과 CSS는 프로그래밍 언어가 아니다!

Cascading Style Sheet

Hyper Text Markup Language
하이퍼 텍스트가 무슨 뜻인지는 중요치 않다

Mark-up 언어라는 것이 중요한데,
예를 들면 책을 읽고 있는데 이 부분이 중요하다고 밑줄을 긋는 것과 같다.

그게 HTML이다.

어느 부분이 타이틀, 어디가 링크 부분인지 마킹하는것.
마킹하는 언어
각각 무엇인지 알려주는 언어

HTML은 브라우저에게 이건 제목, 이건 링크, 이건 리스트, 이건 헤더 등등을
다 HTML로 알려주는 것이다.


CSS는 브라우저에게 각각 요소들이 어떻게 생겨 먹어야하는지 알려줌

HTML이 이건 링크고 이건 이미지야 라고 알려준다면
CSS는 저 텍스트는 빨간색이고 저 이미지는 커야하고, 저 네비는 옆에 있어야하고 등등을 알려주는 것.

이게 HTML과 CSS의 차이점 그리고 서로 보완.

HTML - 각 요소가 무엇인지 설명
CSS - 각 요소의 색상, 크기, 배경은 어떠한지 설명

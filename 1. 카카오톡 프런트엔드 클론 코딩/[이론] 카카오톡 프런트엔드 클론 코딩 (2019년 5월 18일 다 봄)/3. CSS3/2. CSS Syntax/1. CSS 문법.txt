CSS는 2가지 파트로 구성.

첫번째는 selector

아래의 property를 HTML의 h1태그에 연결하고 싶다면??

예시. 

h1 {
   property-name: value;
}

위의 괄호 사이에는 수많은 property를 원하는 만큼 넣을 수 있다.
괄호 사이에 넣는 모든 property가 h1에 적용될 것.

h1{ } 가 selector인데,  여기에는 태그, 링크, 문단, 헤더, 푸터가 될 수 있다.
즉, 모든 태그 이름이 해당된다. 물론 div도 가능

★ ID나 Class도 쓸 수 있음
ID의 경우 #이랑 같이 작성해야함
Class의 경우에는 .이랑 같이 작성해야함

또한 섞을 수도 있다. 
h1 .name { } 이런식으로   태그 + Class


두번째는 property

예시. property-name: value;

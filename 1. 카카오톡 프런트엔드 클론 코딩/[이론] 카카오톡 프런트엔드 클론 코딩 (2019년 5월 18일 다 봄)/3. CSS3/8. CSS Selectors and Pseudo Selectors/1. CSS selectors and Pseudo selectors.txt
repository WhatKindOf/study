기존에 배운 CSS selectors

태그에 class명 지어주고
style태그에서 .클래스명


새로운 Pseudo selector (가상 셀렉터) - 셀렉터인데 element가 아닌 것.

예를 들어

<input type="submit"> 이라고 있으면
기존의 CSS selector는 <input type="submit" class="submit">
이렇게 클래스를 주고 

<style>
	.submit{
	}
</style>

했을 텐데

pseudo selector (가상 셀렉터)는

<style>
	input[type="submit"]{
	}
</style>

이렇게 할 수 있다.

위처럼 [type=""] 해서 input태그 중에서도 type에 따라 다르게 할 수도 있고
아니면 모든 input태그에 대해서 적용할려면

<style>
	input{
	}
</style> 

이렇게 하면 된다.


만약 여러 박스 중에서 첫번째나 마지막 혹은 n번째 있는 박스만 
다른 색칠할려면 어떻게 해야할까

.box:nth-child(숫자){
}
숫자에 n번째에 해당하는 숫자를 넣으면 해당 번째에 있는 박스만 설정가능

.box:first-child{}

.box:last-child{}

위처럼 첫번째나 마지막도 가능


만약 박스가 매우 많다면
.box:nth-child(2n){}
위처럼 작성하면
2의 배수에 있는 것들만 적용 가능하다

3n+1 이나 2n 처럼 수식을 넣을 수 있다는 것.


이 외에도

input + .box{
}
input과 box가 형제라는 뜻이며

input > .box{
}
input 아래에 있는 .box
즉, 직계를 뜻함

input 밖의 box는 해당되지 않는다

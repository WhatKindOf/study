Array - 데이터를 저장하는 곳, 리스트같이 저장. 여러 string들을 하나로 묶을 수 있다.

Object


※ 변수명을 지을 때 camel case를 지켜야 함.
Camel case : 언제나 소문자로 시작해서 
만약에 변수명 중간에 스페이스가 필요하면 
스페이스 대신에 대문자를 써줘야함.  (스페이스는 쓸 수 없다)

항상 소문자로 시작하고 스페이스가 필요하면 다음 단어 첫글자를 대문자로!



Array를 만드는 방법?
const daysOfWeek = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun", true];
[ ]안에 넣으면 (string, boolean, number, float) 그게 Array가 된다.
(같은 데이터 타입을 넣지 않아도 된다. 오! 심지어 변수를 넣어도 된다.)

Array의 규칙?
Wed를 보고 싶다면
daysOfWeek[2]를 사용하면 됨.
3으로 안 하는 이유는 0부터 시작하기 때문.

미니노트, 코멘트(주석)

주석처리되지 않은 모든 코드는 실행하려고 함.

// 이 기호는 '이 기호 뒤에 작성하는 것은 실행하지마' 라고 알려주는 것

// 는 한줄 처리 주석

/* */ 는 멀티라인 주석


변수에 담을 수 있는 데이터 타입에 대해 얘기해보자.

우선 변수를 선언할 때는 기본적으로 const를 쓰도록.
진짜 필요할 때만 let을 쓰도록 하자.


1. String (보통 텍스트 스트링)
const what = "?????"  이렇게 하면 색상이 지정된다.

const what = Nicolas 
이렇게 ""를 제거하게 되면 
프로그램은 Nicolas라는 키워드를 찾으려고 한다.
근데 찾을 수 없으니 에러가 발생한다.

ReferenceError : Nicolas is not defined

텍스트 스트링을 넣을려면 const what = "Nicolas" 
이렇게 ""를 붙여줘야 한다.

텍스트에는 이모티콘도 가능하다.

"" 사이에 숫자를 넣으면 그 변수에는 숫자가 담긴게 아니라
텍스트가 담긴 것이다.

2. Boolean (true or false)

const what = true;
const wat = false;

모두 소문자로 쓰고 "" 없이 쓴다.
바이너리(이진법)에서 모든 것은 0과 1이다

true는 1이고 false는 0이다.


3. Number (숫자)
많이 다뤄봤으니 넘어감.


4. Float (떠돌이 소수점을 가진 숫자)

const wat = 55.1;

Object와 Array의 다른 점은

Object에는 각 value에 이름을 줄 수 있다.


만약 당신이 당신의 개인 정보를 저장하고자 한다면
Array로 이를 저장하고자 한다면 비효율적.

const privateInfo = ["name","gender","age"];

그래서 Object로 저장하도록 하자.

Object 생성방법

const privateInfo = { 
	name:"Nico",
	age:33,
	gender:"Male",
	isHandsome:true	
}

"name"이라 쓰지않고 그냥 name이라고 쓴다
왜냐 name이라는 변수니까
각 변수끼리의 구분은 , 쉼표로 해준다.

만약 gender에만 접근하고자 한다면 다음과 같이

privateInfo.gender 

이렇게 써주면 된다.

그리고 각 변수의 값을 바꾸고 싶다면
privateInfo.gender = "Female"; 

흥미로운 점은 const로 privateInfo를 생성했는데,
privateInfo 안에 있는 변수의 값을 바꿀 수 있다.

하지만 privateInfo = true; 와 같이 privateInfo 자체를 바꿀 순 없다.


데이터를 정렬하는 방법은 두 가지
1. Array    2. Object

만약, DB에서 가져온 리스트 데이터라면,
Array를 선택하고,
근데 만약 데이터를 합쳐서 만들어야 한다면

Object 또한 Array 안에 넣을 수 있다.

Object 안에 배열을 넣을 수도 있다.

const nicoInfo = {
	name:"Nico",
	age:33,
	favMovies: ["1","2","3"]
}

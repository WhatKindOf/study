JavaScript는 단지 HTML과 CSS를 바꾸고 싶어서 만들어지진 않음

이벤트에 반응하기 위해서 만들어짐

그럼 이벤트란 무엇인가 - 웹사이트에서 발생하는 것들.
click, resize, submit, input, change, load 같은 것들

우리는 이 이벤트를 중간에 가로챌 수 있다.


window.addEventListener("resize", 우리가 이벤트에서 다룰 함수)
이벤트를 받기를 기다린다. (listen to event)
어떤 이벤트를 기다리는지 명확하게 해줄 것.


function handleResize( ){
	console.log("dddddd");
}

window.addEventListener("resize", handleResize);
이벤트를 받으면(윈도우 사이즈가 변경되면) handleResize함수를 호출.

※ 주의할 점.
handleResize라고 작성하지 않고 
handleResize( )라고 작성하게 되면 
원하는 시기가 아니라 지금 바로 호출하라는 뜻이므로
조심할 것!!!!!

const title = document.querySelector("#title");

function handleClick() {
  title.style.color = "blue";
}

title.addEventListener("click", handleClick);

이번엔 위처럼 title이 클릭되면 색상이 바뀌도록 해보았다.

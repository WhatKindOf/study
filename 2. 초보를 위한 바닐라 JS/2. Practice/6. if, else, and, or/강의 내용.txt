이번 시간에 할 내용은 if-else.

if(조건){
 block
} else {
 block
}


간단하게 다음과 같이 해볼 수 있다.
if(10 === 5) {
	console.log("hi");
} else {
	console.log("ho");
}

변수에 값을 대입할 때는 = 하나만 쓰지만
무언가 두 개의 값이 같을지 검사할 때는 === 3 개를 써야한다.

if(20 > 5 && "nicolas" === "nicolas") {
	console.log("hi");
} else {
	console.log("ho");
}

위 처럼 &&를 이용해서 여러 조건을 검사할 수도 있다.
&&은 왼쪽과 오른쪽 둘 다 참이어야 바로 아래의 block을 실행하고
하나라도 거짓이면 else문의 block을 실행.

|| 이 것은 or인데 여러 조건 중 하나만 참이면 된다.
둘 다 거짓이어야 else문의 block이 실행된다.
